# CoffeeGuard AI

CoffeeGuard AI classifies Arabica coffee leaf images into healthy, rust, leaf miner, brown leaf spot, and Cercospora leaf spot classes.

This app is a decision-support tool for education and agronomy triage. It is not a definitive agricultural diagnosis.

Model: EfficientNet-B0 fine-tuned on the BRACOL Leaf Dataset.
