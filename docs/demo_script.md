# 1.5-Minute Demo Script

0:00-0:15 - Introduce CoffeeGuard AI and the coffee leaf disease problem.

0:15-0:45 - Open the deployed Gradio app, upload a coffee leaf image, and show the predicted disease and confidence.

0:45-1:10 - Upload two more disease examples and point out the top-3 probabilities and Grad-CAM explanation.

1:10-1:25 - Show the GitHub repository and deployed Hugging Face Spaces link.

1:25-1:30 - State that the app is a decision-support tool and not a replacement for expert diagnosis.
